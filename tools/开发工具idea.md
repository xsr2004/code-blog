**查看所有实现类图**

```cobol
某个接口上 -> Show Diagrams -> 选中某个接口，点右键 -> Show Implementations -> Ctrl + A ->  Enter 
```

**查看所有实现类结构** 

```cobol
在某个接口或者类上按ctrl+h
```
